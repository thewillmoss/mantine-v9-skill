# @mantine/schedule Reference

New package in Mantine 9.x for calendar scheduling views.

```bash
npm install @mantine/schedule
```

Import styles:

```tsx
import '@mantine/schedule/styles.css';
```

## Components

### Schedule

Unified container combining all views with built-in navigation.

### DayView

Single day display with configurable time slots and current time indicator.

### WeekView

Weekly calendar grid with week numbers and multi-day event spanning.

### MonthView

Monthly grid with event overflow handling and week numbers.

### YearView

12-month overview organized by quarters.

### MobileMonthView

Mobile-optimized month view with event details panel.

For full API details, refer to the Mantine documentation at mantine.dev.
